# KeyManager
KeyManager is an Admidio plugin for managing building and room keys.

### Installation and Update

1. Create a folder named 'KeyManager' under adm_plugins
2. Copy all plugin files into this folder (If the folder already exists, you only need to replace all the files within it.)
3. To install (or update) , run the following PHP file: .../adm_plugins/KeyManager/system/install.php
